# 📌 Descripción

**KULLAKA** es una plataforma Ecommerce orientada a la venta online de indumentaria y accesorios en **Salta Capital**, creada como un emprendimiento familiar con presencia física en espacios de emprendedores como **Wonder**.

El proyecto busca ofrecer una experiencia moderna, rápida y segura para la compra online de ropa y accesorios, integrando autenticación de usuarios, carrito de compras y una interfaz responsive enfocada en la experiencia del cliente.

# ⚙️ Instalación y Uso

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/tuusuario/kullaka-ecommerce.git
```

## 2️⃣ Entrar al proyecto

```bash
cd kullaka-ecommerce
```

## 3️⃣ Instalar dependencias

```bash
npm install
```

## 4️⃣ Ejecutar entorno local

```bash
npm run dev
```

---

# 🔥 Variables de Entorno

Crear archivo `.env`

```env
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```
